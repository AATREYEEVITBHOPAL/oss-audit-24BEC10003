About Project
This repository contains all five shell scripts made for the Open Source Audit project.
I have chosen Python because it is simple, widely used, and fully open source.
It is maintained by the Python Software Foundation and follows the PSF License.
________________________________________
Scripts Overview
Script 1 – System Identity Report
This script shows basic system details like:
•	Linux distribution
•	Kernel version
•	Current user
•	Uptime
•	Date and time
•	License info
________________________________________
Script 2 – Package Inspector
This script checks if Python is installed or not.
It also shows version details and prints a small description using a case statement.
________________________________________
Script 3 – Disk and Permission Auditor
This script checks important system folders like:
•	/etc
•	/home
•	/var/log
It shows:
•	Folder size
•	Permissions
•	Owner
________________________________________
Script 4 – Log File Analyzer
This script reads a log file and counts how many times a keyword appears.
Example: error, warning, install
________________________________________
Script 5 – Manifesto Generator
This script asks 3 simple questions and creates a small open-source message.
It saves the output in a text file.
